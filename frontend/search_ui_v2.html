OTA AI Search - Real Search / Context Demo

Giao diện này không dùng dữ liệu giả. Search gọi backend thật qua POST /search, khi mở chi tiết khách sạn sẽ gọi POST /context. Trace nâng cao có thể đọc từ GET /hybrid_search khi backend sẵn sàng.

Scope hiển thị Hiếu hiển thị output của DA10: Top-K results, metadata, citations/source ids, context ids, evidence và LLM context. Retrieval/ranking/evaluation calculation thuộc backend.
Nhập query hoặc chọn query mẫu. Backend cần chạy tại http://localhost:8000.
