DA10 Explainable Retrieval Console

Prototype giao diện frontend kết nối backend thật. Console này không sửa backend và không giả lập metric retrieval.

Cấu trúc giao diện

  • Ô nhập query
  • Kiểm tra backend
  • Hiểu query và ontology tag
  • Baseline BM25
  • Luồng retrieval
  • Giải thích xếp hạng
  • Context package
  • Bằng chứng review/citation
  • TODO: độ tin cậy và số lượng từng bước nếu backend expose thêm.

API được gọi

  • GET /health: kiểm tra API đúng service hay không.
  • GET /hybrid_search: luồng explainability chính.
  • GET /search: baseline BM25.
  • POST /context: evidence của khách sạn được chọn.

Sơ đồ nhanh

[Query + API URL + Health] | +-- Hiểu query + ontology tags +-- Baseline BM25 | +-- Luồng retrieval | candidates -> BM25 -> vector -> fusion -> rerank | +-- Giải thích xếp hạng | final_score / business / rrf / rerank | +-- Context + evidence chunks / citations / review evidence

Luồng xử lý query

Mục tiêu: cho thấy một câu hỏi được hiểu, gắn tag ontology, truy vấn BM25, rồi đi tới danh sách khách sạn.

FALLBACK KHI HYBRID TIMEOUT
Chạy query để xem luồng xử lý.

A. Hiểu query và ontology tags

Fields: hybrid_search.intent.*

SẴN SÀNG NẾU BACKEND TRẢ FIELD
Chạy query để xem concepts, intent, location và expansion.

Baseline BM25

Fields: search.results[].score, total_hits, took_ms

DỮ LIỆU THẬT
Chạy query để xem baseline BM25/OpenSearch.

B. Luồng retrieval

Fields: n_candidates, n_fused, bm25_rank, vector_rank, rrf_score, rerank_score

PARTIAL: chờ số lượng từng bước
Chạy query để xem retrieval pipeline.

C. Giải thích xếp hạng

Click một khách sạn để gọi POST /context nếu backend hỗ trợ.

SẴN SÀNG KHI HYBRID TRẢ TOP HOTELS
Chạy query để xem bảng ranking.

D. Context package

context_package.chunks[] (chunk_id + nội dung + metadata) · prompt/llm_context rút gọn tối đa 200 ký tự

CHỌN KHÁCH SẠN
Chọn một khách sạn trong bảng ranking để xem context.

E. Bằng chứng review/citation

Fields: evidence.positives[], negatives[], citations[]

CHỌN KHÁCH SẠN
Chọn một khách sạn để xem evidence/citations nếu backend trả về.

F. Đánh giá Golden Dataset

Chạy golden_set_v2 qua pipeline → Recall / Precision / Hit / MRR / nDCG. Nguồn: GET /eval/golden.

METRIC THẬT
Bấm "Chạy đánh giá" để đo metric trên golden dataset. Lưu ý: nhiều câu sẽ lâu (~2s/câu).

SOURCE OF UI
Khu vực UI Endpoint Field sử dụng
